cryptid_randomBytes

CryptidStatus cryptid_randomBytes(
  unsigned char *buf,
  const int num
)

Fills the passed buffer from a cryptographically secure source. If {@code (__CRYPTID_EXTERN_RANDOM} is defined, then this function will call the {@code int __cryptid_cryptoRandom(void *buf, const int num)} function internally.

Parameters

*buf

unsigned char

num

const int

The size of the buffer.

Return Value

CryptidStatus

CRYPTID_SUCCESS if everything went right.
