random_solinasPrime

CryptidStatus random_solinasPrime(
  mpz_t result,
  const unsigned int numberOfBits,
  const unsigned int attemptLimit
)

enerates a cryptographically secure random Solinas prime having the specified bitlength. A Solinas prime is a prime of the form \f$2^a \pm 2^b \pm 1\f$, where \f$a > b\f$.

Parameters

result

mpz_t

Out parameter for the generated value.

numberOfBits

const unsigned int

The bitlength of the result.

attemptLimit

const unsigned int

The maximum number of attempts before giving up the generation.

Return Value

CryptidStatus

CRYPTID_SUCCESS if everything went right.
